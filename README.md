# 💰 Expi-Tracker

A modern, full-featured personal finance tracker built with React + Tailwind CSS. Manage your income and expenses with a clean fintech-style dashboard.

## 🌐 Live Demo
[View on Vercel](https://expi-tracker.vercel.app)

## ✨ Features

- 🔐 **Authentication** — Signup & Login with localStorage-based credential validation
- 📊 **Dashboard** — Real-time balance, income, expense summary cards
- 📈 **Analytics** — Area charts, bar charts, pie charts, category breakdown with progress bars
- 💸 **Transactions** — Add, delete, search, and filter by category or type
- 🌗 **Light / Dark Mode** — Toggle from the navbar, preference saved across sessions
- 🇮🇳 **INR Currency** — All amounts displayed in Indian Rupees (₹)
- 📱 **Fully Responsive** — Works on mobile, tablet, and desktop
- 💾 **Local Storage** — All data persists in the browser, no backend needed
- 🎨 **Premium UI** — Glassmorphism cards, smooth animations, custom dropdowns
